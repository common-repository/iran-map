== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/iran-map` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Copy this shortcut [display_iran_map] to page where you want to display map
